## Setup & Start

Alle gezeigten Kommandozeilen-Befehle setzen voraus, dass [Node.js](https://nodejs.org/) installiert ist. 

```cmd
$ cd src
$ npm install
$ npm install -g jspm
$ jspm install
$ npm start
```

Nach `npm start` öffnet sich der Standard-Browser.
